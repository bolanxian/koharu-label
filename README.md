# Koharu Label
将`发声时机`和`音高`组合为`Synthesizer V Studio`工程文件。

## [下载版](https://pan.baidu.com/s/1fJgz6Resv2gt_GJaLMw6Tw?pwd=khrr)
- 将`koharu-label.7z`和`koharu-label-runtime.7z`解压到同一目录
- 运行`run.bat`
- 运行环境：Windows 10 x64+现代浏览器

## 手动构建
```
npm install
npm run build
```
**Python依赖**
```
npm run requirements
```
**启动**
```
py -3 app.py
```
