### 从 NEUTRINO

`NEUTRINO/output/*.f0`  

### 从 VOICEVOX
首先在 设置＞保存 打开 `labファイルを書き出し`  

### 关于 PyWORLD
PyWORLD是WORLD声码器的Python包装。  
WORLD是高质量语音分析、操作和合成系统。  
可以估计基频(F0)、频谱包络、非周期性指数，  
也可以仅使用估计参数合成语音。  
WORLD声码器提供了 `DIO` 和 `Harvest` 两个基频估计方法  
在本页面控制台执行 `vm.useHarvest=true` 可切换为 `Harvest`
